# 3660. [Jump Game IX](./3660.cpp)

쩜-프

왼쪽으로 점프하려면 자신보다 큰 수로만 갈 수 있고, 오른쪽으로는 자신보다 작은 수로만 점프할 수 있다.

일종의 방향 그래프가 나올 것 같지만, 오른쪽으로 점프를 뛴다면 점프를 마친 인덱스에서 다시 원래 인덱스로 점프를 할 수 있을 것이다. 그래서 방향이 없는 연결 그래프가 된다.

그럼 그 연결이 어떻게 이루어질지 고민을 해보자.

구간 $[0, i]$ 와 구간 $[i + 1, n - 1]$ 에 대해 왼쪽 구간의 최대값을 $P_MAX$, 오른쪽을 $S_MIN$ 이라고 하자.

그러면 왼쪽 구간에서 $P_MAX$ 가 등장한 위치부터 $i$ 까지는 전부 $P_MAX$ 로 채울 수 있음이 자명하다.
