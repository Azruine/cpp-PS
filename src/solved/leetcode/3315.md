# 3315. [Construct the Minimum Bitwise Array II](./3315.cpp)

제일 작은 수를 원하니까, 제일 큰 비트를 뒤집어야 한다.

그러면 어떤 비트를 뒤집어도 될까? 답은 LSB부터 연속한 1이 끝나는 비트다.

101111 을 예로 들면, 1 << 3 을 제거하는게 최적이라는 뜻이다.

이제 그걸 최대한 빠르게 구해보자. 비트 연산만으로 구해낼 수 있다.

```cpp
auto x = (~elem & (elem + 1)) >> 1;
```

이게 바로 우리가 원하는 1 << 3 을 한 방에 구하는 코드다. 맙소사.

아무튼, 구해진 친구가 1이라면, 즉 짝수라서 연속한 1이 없었다면? mask를 0 - 1 로 만들어서 모든 비트를 켜준다.

그렇지 않으면 mask가 0이 된다.

마지막으로 mask랑 elem ^ x를 bitwise OR 연산을 해주면 우리가 원하는 답이 나온다.

주석 처리는 체이닝을 활용한 한 줄 코드다. 끔찍하네.
